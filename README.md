# Learn2SolveSudoku
Python code to solve sudoku for educational purpose. This repository contains the template for the students

This is for the fourth lecture on "Algorithms and Data Structures" at a Kantonsschule (Swiss version of high school).
The students are in the second year, they already had one year lectures in "Informatik". 
The first three lectures of the Algorithms and Data Structures theme covered Karatsuba multiplication, Euclid's extended algorithm, 
and hash tables. 

The intent for the lecture on sudoku is to divide the class into teams, and each team has to implement  one or two functions 
with the complexity of one loop and several conditions. 
The comments are in German.

The full solution will be in another - non-public - git repo.
